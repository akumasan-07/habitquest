# HabitQuest

HabitQuest is a personal productivity and habit tracking web application built with React and Tailwind CSS.

## Features

* Create and manage daily quests (habits)
* Manage quest completion for today and yesterday
* View productivity score
* Track streaks and consistency
* Visualize progress using a heatmap
* Light and dark theme support
* Local storage persistence

## Tech Stack

* React
* React Router
* Tailwind CSS
* date-fns

## Future Plans

* Express backend
* MongoDB Atlas integration
* User authentication
* Gamification system
* Advanced analytics

## Running Locally

```bash
npm install
npm run dev
```
